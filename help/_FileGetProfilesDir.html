Function Reference

_FileGetProfilesDir

Gets the local profiles directory from the registry.

_FileGetProfilesDir ( )

 

Parameters

None.

 

Return Value

Success - Returns the full path of the local profiles directory.
Failure - Returns "" and sets @error = 1.

 

Remarks

None.

 

Related

RegRead (see AutoItV3 documentation).

 

Example


#include "FileLib.au3"
$Result = _FileGetProfilesDir()

;$Result = "C:\Documents and Settings"
