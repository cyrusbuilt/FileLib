Function Reference

_DiskIsNetwork

Determines if a specified drive letter is a network drive.

_DiskIsNetwork ( Drive )

 

Parameters

Drive A logical drive letter followed by a colon. i.e.: "D:"

 

Return Value

True - Specified drive letter is a network drive.
False - Specified drive is not a network drive.
    Sets @error:
       1 - if drive does not exist
       2 - if unable to determine drive type.

 

Remarks

None.

 

Related

_DiskExist, _DiskInfoToArray

 

Example


#include "FileLib.au3"

If _DiskIsNetwork("X:") = True Then
   MsgBox(0, "","Drive X: is a network drive.")
EndIf
