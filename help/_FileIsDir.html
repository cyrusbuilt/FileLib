Function Reference

_FileIsDir

Determines whether or not the supplied path is a directory.

_FileIsDir ( Path )

 

Parameters

Path The path to check.

 

Return Value

True - The specified path is a directory.
False - The specified path is not a directory. Sets @error = 1 if "Path" does not exist.

 

Remarks

_FileCheckDirEmpty

 

Related

None.

 

Example


#include "FileLib.au3"

If _FileIsDir(@ScriptDir) Then
   MsgBox(0, "", @ScriptDir & " is a directory.")
EndIf
