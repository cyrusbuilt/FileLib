Synopsis

CyrusBuilt File Utilities Library for AutoIt V3 was created to "scratch an itch". This library is essentially a collection of functions I initially wrote in other scripts due to the lack of a similar function (or a similar function existed but didn't operate the same way) being built into AutoItV3. Since I started using these functions frequently in other projects, it was decided to build a seperate library so that I could just include it in future projects.

The purpose of this library is to provide certain file/disk facilities that don't currently exist in AutoIt such as disk-existence checking or recursive path listing.

Features:

  • Get library version
  • Get File/Folder extension
  • File Comparison
  • Disk (drive letter) existence checking
  • Get info about a drive (returns an array)
  • Check to see if a drive letter is a network drive
  • Test to see if a path is a directory
  • Recursive full-path listing of a specified directory
  • Get a file name from a path
  • File/Directory copy with progress dialog
  • Move File(s)/Directory with progress dialog
  • Delete File(s)/Directory with progress dialog
  • Rename a file or directory with progress dialog
  • Check if a directory is empty
  • Get the directory of a specified file path
  • Truncate a file name to DOS style 8.3 format.
