Function Reference

Below is a complete list of the functions available in the CyrusBuilt File Utilities library.  Click on a function name for a detailed description.

 

Function Description
_GetFileLibVersion Gets the current version of this library.
_FileExtension Returns the file extension of the specified path.
_FileCompare Compares 2 files to see if they are the same.
_DiskExist Checks to see if a drive letter exists.
_DiskInfoToArray Returns an array containing info about a specified drive letter.
_DiskIsNetwork Determines if a specified drive letter is a network drive.
_FileIsDir Determines whether or not the supplied path is a directory.
_FileRecurseBuildList Performs file recursion on the specified directory and builds an array containing the full path to every file in the specified directory and every subdirectory.
_FileGetFileName Gets the file name from the specified file path.
_FileGetDirFromPath Returns the directory where a specified file (full path) exists.
_FileCopyDlg Copys a file, multiple files, or directory to the specified destination.
_FileMoveDlg Moves a file, multiple files, or directory to the specified destination.
_FileDeleteDlg Deletes a file, set of files, or directory while displaying a progress dialog.
_FileRenameDlg Renames a file or directory and displays a progress dialog.
_FileCheckDirEmpty Checks to see if a specified directory is empty.
_FileTruncateName Returns a truncated file name in 8.3 (8 character name, 3 character extension) DOS style format.
_FileGetSystemDrive Returns the logical drive letter of the partition Windows is installed on.
_FileGetProfilesDir Returns the full path of the local profiles directory.
_FileRemoveDirIfEmpty Removes a specified directory only if it is empty.
