Function Reference

_FileCompare

Compares 2 files to see if they are the same.

_FileCompare ( File1, File2 )

 

Parameters

File1 First file to compare.
File2 Second file to compare.

 

Return Value

True - Files are the same.
False - Files are different.
Sets @error to:
1 - File1 not defined, is a directory, or does not exist.
2 - File2 not defined, is a directory, or does not exist.
3 - Files did not match or 'FC' command error and sets @extended to the ERRORLEVEL returned by 'FC'.

 

Remarks

None.

 

Related

None.

 

Example


#include "FileLib.au3"

If _FileCompare("SomeFile.txt", "SomeFile_old.txt") Then
   MsgBox(0, "", "Files are the same.")
EndIf
