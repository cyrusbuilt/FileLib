Function Reference

_FileExtension

Gets the file extension of the specified filed path.

_FileExtension ( "path" )

 

Parameters

path The name or full path of the file to get the extension of.

 

Return Value

Success - Returns the extension of the file specified without the dot (".").
Failure - If the specified name or path does not have an extension it will return an empty string. If $Path is empty or undefined then an empty string is returned and @error is set to 1.

 

Remarks

None.

 

Related

None.

 

Example


#include "FileLib.au3"
$Extension = _FileExtension("myfile.txt")
;$extension = "txt"
