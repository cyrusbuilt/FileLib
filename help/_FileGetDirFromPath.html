Function Reference

_FileGetDirFromPath

Returns the directory where a specified file (full path) exists.

_FileGetDirFromPath ( "path" )

 

Parameters

path A full path to a specific file.

 

Return Value

Success - Returns the path of the file without the filename.
Failure - Returns an empty string and sets @error to one of the following:
1 - The value specified for path is an empty string or not a string at all.
2 - The specified path does not contain a valid path separator.

 

Remarks

None.

 

Related

_FileIsDir, _FileGetFileName

 

Example


#include "FileLib.au3"

$Result = _FileGetDirFromPath("C:\temp\myfile.txt")
;$Result = "C:\temp\"
