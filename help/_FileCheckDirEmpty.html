Function Reference

_FileCheckDirEmpty

Checks to see if a specified directory is empty.

_FileCheckDirEmpty ( Path )

 

Parameters

Path A full directory path (MUST be a directory).

 

Return Value

Success - True. Directory is empty.
Failure - False. Sets @error to 1 if $Path is not a directory or does not exist. Sets @error to 2 if unable to establish a search handle for $Path.

 

Remarks

None.

 

Related

_FileIsDir

 

Example


#include "FileLib.au3"

If _FileCheckDirEmpty("C:\temp") = False Then
   MsgBox(0, "Dir Check", "Directory not empty!")
EndIf
