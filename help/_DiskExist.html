Function Reference

_DiskExist

Checks to see if a drive letter exists.

_DiskExist ( Drive )

 

Parameters

Drive A logical drive letter followed by a colon. i.e.: "D:"

 

Return Value

True - Drive letter exists and is valid.
False - Drive does not exist. Sets @error = 1 if parameter undefined.

 

Remarks

None.

 

Related

_DiskInfoToArray, _DiskIsNetwork

 

Example


#include "FileLib.au3"

If _DiskExist("C:") Then
   MsgBox(0, "", "Drive C: is present.")
EndIf
