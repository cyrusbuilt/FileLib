Function Reference

_DirMakeSymlink

Creates a symlink to specified target directory.

_DirMakeSymlink( linkPath, targetPath )

 

Parameters

linkPath The path to the link to be created.
targetPath
The path to the actual directory to link to.

 

Return Value

Success - True
Failure - False and sets @error to one of the following:
     1 - linkPath is an empty string.
     2 - targetPath is an empty string.
     3 - linkPath already exists and is a symlink.
     4 - targetPath is not a directory or does not exist.
     5 - Failed to create the link at the specified path.

 

Remarks

None.

 

Related

_DirIsSymlink

 

Example


#include "FileLib.au3"
$Result = _DirMakeSymlink("\MyDocs", "C:\Users\User1\Documents")
;$Result = True
