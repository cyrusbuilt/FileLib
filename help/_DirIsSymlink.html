Function Reference

_DirIsSymlink


Checks to see if the specified path is a symlink (reparse point).

_DirIsSymlink( dirPath )

 

Parameters

dirPath The directory path to check.

 

Return Value

Success - True
Failure - False if path is empty, does not exist, or is not a symlink.

 

Remarks

None.

 

Related

_DirMakeSymlink

 

Example


#include "FileLib.au3"
$Result = _DirIsSymlink("\MyDocs")
;$Result = True
