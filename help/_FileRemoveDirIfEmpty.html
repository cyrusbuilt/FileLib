Function Reference

_FileRemoveDirIfEmpty

Removes the specified directory only if it is empty.

_FileRemoveDirIfEmpty ( "dirPath" )

 

Parameters

dirPath The directoy path to check and remove.

 

Return Value

Success - Returns True.
Failure - Returns False if there is an error removing the directory, directory does not exist, or directory is not empty.

 

Remarks

None.

 

Related

_FileCheckDirEmpty
DirRemove

 

Example


#include "FileLib.au3"
$path = @TempDir & "\child_temp"
DirCreate($path)
$result = _FileRemoveDirIfEmpty($path)

; $result == True
