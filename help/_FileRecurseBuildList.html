Function Reference

_FileRecurseBuildList

Performs file recursion on the specified directory and builds an array containing the full path to every file the specified directory and every subdirectory.

_FileRecurseBuildList ( Path, ByRef aFiles )

 

Parameters

Path The path to check.
ByRef aFiles An array to contain the file paths and then count.

 

Return Value

Success - aFiles - The resulting path array with the path count at element 0. If no file or directory exists within the parent path specified then $aFiles will be returned with only element 0 and it's value will be 0.

Failure - If the parent path specified does not exist, the same will be true but it will also set @error to 1.

 

Remarks

None.

 

Related

_FileIsDir

 

Example


#include "FileLib.au3"
#include <Array.au3>

Dim $aResult[1]
$aResult[0] = 0
$aResult = _FileRecurseBuildList(@ScriptDir, $aResult)
_ArrayDisplay($aResult)
